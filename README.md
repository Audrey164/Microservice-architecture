# Node.js Microservices Architecture

A modern microservices architecture using Node.js for scalable enterprise systems.

## Services Included
- Auth service  
- Product service  
- Payment service  
- Notification service  
- Gateway API  

## Stack
- Node.js  
- Express  
- MongoDB  
- Docker  
- Nginx  

## Installation
docker-compose up -d --build
